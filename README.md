# Sistema de Medicao

Sistema web para cadastro, acompanhamento e exportacao de medicoes e relatorios fotograficos.

## Recursos

- Dashboard quantitativo.
- Registros editaveis e organizados em caixas.
- Status de atividades concluidas ou em espera.
- Exportacao de relatorios em XLSX.
- Geracao de relatorio com envio para `comercial1@primecsg.com.br`.

## Executar localmente

```powershell
npm install
npx vercel dev
```

## Verificacao

```powershell
npm run check
```
